INSTRUCTIONS
============

1. Put the java package folder 'assignment4' in a location with write permission, for example, the root folder in C: drive. So the path for the folder would be -

C:\assignment4

2. Open Command Prompt from the start menu in windows.

3. Go to the folder location where the 'assignment4' folder is located. For this example, type -

C: (to change the drive)
cd/ or cd\ (to go to root)

4. (Optional) To recompile, enter the folloing command -

C:\javac assignment4/SortTest.java

4. To run the application class, enter the following command -

C:\java assignment4/SortTest

5. Follow the on-screen instructions for user inputs.
