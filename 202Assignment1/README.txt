INSTRUCTIONS
============

1. Put the java package folder 'assignment1' in a location with write permission, for example, the root folder in C: drive. So the path for the folder would be -

C:\assignment1

2. Open Command Prompt from the start menu in windows.

3. Go to the folder location where the 'assignment1' folder is located. For this example, type -

C: (to change the drive)
cd/ or cd\ (to go to root)

4. (Optional) To recompile, enter the folloing command -

C:\javac assignment1/UseMyBigDecimal.java

4. To run the application class, enter the following command -

C:\java assignment1/UseMyBigDecimal

5. Follow the on-screen instructions for user inputs.

6. To compare/validate the calculation results, check out this website: http://comptune.com/calc.php
